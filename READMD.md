Description


The Banking Card System is a Python-based console application designed to help users manage digital money transactions securely and efficiently. The system includes login authentication, balance checking, money conversion between PHP and USD, cash-in services, and money transfer using the Padala Pay feature. It also generates digital receipts and records transaction details automatically.

How the System Works


1. The user enters a valid username and password to access the system.

2. After successful login, the system displays the main menu options.

3. Users can check their balances in PHP and USD.

4. The money conversion feature allows users to convert currencies using the given exchange rates.

5. The cash-in feature adds money to the selected account balance.

6. The Padala Pay feature allows users to transfer money to another user securely.

7. After every successful transaction, the system generates a digital receipt containing transaction details.

8. The user may continue using the system or exit/logout anytime
